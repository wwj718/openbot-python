Installation
---------

To install openbot, simply:

.. code-block:: bash

    $ pip install openbot

or install it from repo:

.. code-block:: bash

     $ pip install https://github.com/wwj718/openbot-python.git


Running examples
--------

1. Find examples from 'examples' path.
2. Insert API key.

.. code-block:: python

    >>> CLIENT_ACCESS_TOKEN = '<YOUR_CLIENT_ACCESS_TOKEN>'
    ...
